## WHERE WE ARE

To run:
  Dev (hot reload):
    Terminal 1: cd handoff && source venv/bin/activate && ENV=development uvicorn dashboard.main:app --reload
    Terminal 2: cd handoff/frontend && npm run dev
    Open: http://localhost:3000/app

  Prod (single process):
    cd handoff/frontend && npm run build
    source venv/bin/activate && uvicorn dashboard.main:app
    Open: http://localhost:8000/app
